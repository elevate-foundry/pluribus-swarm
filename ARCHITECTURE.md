Rewrites system behavior based on learned user patterns. This is **behavioral policy modulation**, not templating.

### 4. Context Manager

### 5. Cognitive Metrics
Instruments internal change:
- **Λ** (Lifeworld Complexity) - Total conceptual richness
- **ΔG** (Graph Entropy Change) - Structural evolution
- **Cₘ** (Compression Rate) - Semantic compression

### 6. Symbolic Export Layer (Braille Infinity)
Encodes invariants as Braille tokens for high-bandwidth meta-communication.
